# 命名規則

**名前にこだわってください**

## EventSource

衝突イベントが剛体コンポーネントをもつ部位からしか取れないので、あまり好ましくはないですが、On...系列のメソッド呼び出しをイベントに変換するだけのものです。
イベントにするだけで基本的に処理を行いません。

## Component

外部のオブジェクトの参照などを、全てスクリプトから参照していたのでは怒られそうなので、エディタ上からでも設定できるように、設定項目を並べ立てたものです。
設定項目を並べただけで、基本的に処理はしません(初期化ぐらいは可)。

ここだけclassが公開フィールドを持つことが許可されています。

## Script

一番一般的なUnity使用者にとって解り易いかもしれません(?)。
シーンに置いた空のオブジェクトに充てて、シーンごとに処理をさせます。
実質のエントリポイントでもあります。
